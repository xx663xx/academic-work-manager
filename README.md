# Academic Work Manager

Приложение для назначения темы, типа работы и руководителя студентам 3 и 4 курсов.
Проект выполняется в рамках ознакомительной практики. Основная задача - заменить ручное ведение таблиц более продуктивной системой, которая хранит студентов, преподавателей, назначения тем и позволяет выгружать итоговый результат в Excel.

## Основные возможности MVP

- загрузка списка студентов из файла;
- загрузка списка преподавателей из файла;
- поддержка трех ролей
  - администратор;
  - преподаватель;
  - студент;
- выбор студента из списка;
- автоматическое определение типа работы по курсу:
  - 3 курс - курсовая работа;
  - 4 курс - ВКР;
- ввод темы работы;
- выбор преподавателя-руководителя;
- изменение темы работы;
- выгрузка назначений в Excel.

## Планируемый стек

- Python;
- FastAPI;
- Jinja2;

- openpyxl;
- HTML/CSS.

## Запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

После запуска главная страница будет доступна по адресу:

```text
http://127.0.0.1:8000
```

## Результат

Итогом работы приложения должен быть Excel-файл со списком назначений:

- ФИО студента;
- группа;
- курс;
- тип работы;
- тема;
- руководитель;
- статус;
- дата последнего изменения.
