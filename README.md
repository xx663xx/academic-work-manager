# Academic Work Manager

Приложение для назначения темы, типа работы и руководителя студентам 3 и 4 курсов.
Проект выполняется в рамках ознакомительной практики. Основная задача - заменить ручное ведение таблиц небольшой веб-системой, которая хранит студентов, преподавателей, темы, назначения и позволяет выгружать итоговый результат в Excel.

## Что уже реализовано

- веб-интерфейс на FastAPI и Jinja2;
- SQLite-база `academic_work_manager.sqlite3`;
- три роли - администратор, преподаватель и студент;
- импорт студентов и преподавателей из Excel;
- тестовые файлы `samples/students.xlsx` и `samples/teachers.xlsx`;
- справочники студентов и преподавателей;
- создание и изменение назначений администратором;
- автоматическое определение типа работы по курсу:
  - 3 курс - курсовая работа;
  - 4 курс - ВКР или ВКР/курсовая;
- добавление тем преподавателем;
- выбор темы студентом с отправкой на подтверждение;
- подтверждение или отказ по заявке преподавателем;
- редактирование и удаление тем;
- очистка базы данных;
- статусы назначений;
- Excel-выгрузка итоговых назначений;
- схемы в `docs/diagrams/`.

## Стек

- Python 3;
- FastAPI;
- Jinja2;

## Запуск

Из чистой копии проекта выполните команды:

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
.venv/bin/python -m uvicorn app.main:app --reload
```
Также можно использовать `make`:

```bash
make setup
make run
```

После запуска откройте:

```text
http://127.0.0.1:8000
```

## Первичная проверка

Файлы из `samples/` не загружаются автоматически. Для проверки свежего запуска

```text
1. Перейдите в роль администратора.
2. Откройте "Импорт из Excel".
3. Загрузите samples/students.xlsx.
4. Загрузите samples/teachers.xlsx.
5. В роли преподавателя добавьте тему.
6. В роли студента выберите доступную тему.
7. В роли преподавателя подтвердите заявку.
8. В роли администратора откройте назначения и скачайте Excel-файл.
```

## Основные файлы

- `app/main.py` - точка входа FastAPI;
- `app/routes/` - страницы администратора, преподавателя и студента;
- `app/database.py` - SQLite-схема и функции работы с данными;
- `app/excel_io.py` - импорт и экспорт Excel;
- `app/templates/` - Jinja2-шаблоны;
- `app/static/styles.css` - стили интерфейса;
- `samples/` - тестовые Excel-файлы.

## Excel-результат

Выгрузка содержит:

- ФИО студента;
- группа;
- курс;
- тип работы;
- тема;
- руководитель;
- статус;
- дата последнего изменения.
